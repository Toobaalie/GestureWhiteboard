# ✋ Gesture Controlled Virtual Whiteboard

A Computer Vision project that allows users to draw on a virtual whiteboard using only hand gestures. The system tracks hand landmarks in real time and converts finger movements into drawing actions.

## 🚀 Features

- Real-time hand tracking
- Draw using index finger
- Gesture-based interaction
- Smooth virtual whiteboard
- Webcam-based interface
- No mouse or keyboard required

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## 📂 Project Structure

GestureWhiteboard/
│── main.py
│── requirements.txt
│── README.md

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Toobaalie/GestureWhiteboard.git
cd GestureWhiteboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```

## 🎮 How It Works

- Open the webcam.
- MediaPipe detects hand landmarks.
- Raise your index finger to draw.
- Move your finger to create drawings on the virtual canvas.
- Press **Q** to exit.

## 💡 Applications

- Online Teaching
